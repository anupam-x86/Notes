## Another tiny style suggestion

Instead of

```cpp
std::cout << "----------------------------" << '\n';
```

three times,

I'd make it a constant.

```cpp
const std::string_view separator { "----------------------------" };

std::cout << separator << '\n';
```

or

```cpp
constexpr std::string_view separator { "----------------------------" };
```

You'll appreciate this habit in larger programs because if you ever want to change the separator, you do it in one place.
