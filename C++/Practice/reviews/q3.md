# Things I'd improve

## 1. ❌ You didn't fully satisfy the assignment [ Fixed ]

The question says:

> avoid duplicated code where possible

Look carefully.

You duplicated:

```cpp
std::cout << "Temperature in ..."
```

Not a huge deal.

---

## 2. Use constants

The assignment explicitly says

> use constants where appropriate

You didn't.

Instead of

```cpp
return (temperature * 9/5) + 32;
```

I'd probably do something like

```cpp
const double fahrenheitOffset { 32.0 };
const double conversionFactor { 9.0 / 5.0 };
```

Then

```cpp
return (temperature * conversionFactor) + fahrenheitOffset;
```

Why?

Imagine one day the formula changed.

(Obviously temperature formulas don't.)

You'd only edit one place.

More importantly, the names document the meaning of the numbers.

# Tiny style suggestion

Personally I'd write

```cpp
if (userChoice == 1)
```

using

```cpp
constexpr int celsiusChoice { 1 };
constexpr int fahrenheitChoice { 2 };
```

Not because it's necessary...

because later

```cpp
if (userChoice == celsiusChoice)
```

is easier to understand than

```cpp
if (userChoice == 1)
```

Again...

Chapter 5 doesn't require it.
