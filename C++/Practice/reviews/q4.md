### 3. Slightly cleaner `intSign()`

Instead of

```cpp
if (number > 0)
{
    return '+';
}
else if (number < 0)
{
    return '-';
}
return '0';
```

You could write

```cpp
if (number > 0)
    return '+';

if (number < 0)
    return '-';

return '0';
```

Since each branch returns immediately, the `else` isn't necessary.
