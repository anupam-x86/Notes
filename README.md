# Notes

Personal knowledge repository documenting my journey through Linux, programming, computer science, and engineering.


## Purpose

This repository serves as a personal knowledge base where I record:

* What I learn
* Problems I solve
* Commands I use
* Concepts I understand
* Mistakes I make
* Resources I find useful

The goal is not to copy tutorials, but to build my own understanding over time.


## Note Creation Process

These notes are written during my learning journey and are continuously refined
through study, experimentation, project work, discussion, and AI-assisted review.

AI tools are occasionally used to:

- Review technical accuracy
- Refine explanations
- Improve structure and readability
- Identify mistakes, misconceptions, or missing context
- Suggest alternative explanations and examples

> [!IMPORTANT]
> These notes are intended to reflect understanding rather than content generation.
> AI is used to assist learning, challenge assumptions, improve explanations, and review material
> while responsibility for understanding,
> verification, and maintenance remains with me.
